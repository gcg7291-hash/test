# 함수(funtion)

-제어문 -특정 기능을 수행하기 위해 미리 정의한(모아놓은) 코드 블록

## 우리가 사용해온 함수

- `console.log()`
- `String()` : 문자열 형 변환 정의된 함수 (도구)
- `Number()` : 숫자형 형 변환 정의된 함수 (도구)
- `Boolean()`: 불리언 형 변환 정의된 함수 (도구)

## 함수의 장점

- 코드를 재사용하기에 좋다

## 정의(Define) 와 호출(Call)

- 호출 : 함수를 실행 -`함수명()`
- 정의 : 함수의 이름과 기능(코드)를 정의
